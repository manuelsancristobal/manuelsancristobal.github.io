## Resumen del Proyecto
Este análisis busca identificar patrones espaciales y de comportamiento en los atractivos turísticos para mejorar la planificación regional.

### Metodología
- **Técnicas:** Clustering K-Means, Análisis de Densidad (DBSCAN).
- **Herramientas:** Python (Pandas, Scikit-learn), Mapbox.

### Conclusiones
La segmentación reveló tres tipos principales de atractivos, permitiendo una asignación de recursos más eficiente.
